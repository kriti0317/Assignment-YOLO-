# YOLO Object Detection Project

A YOLO-based object detection project using a custom labeled dataset created through image annotation. The model detects and classifies objects in images using deep learning techniques.

---

## 📌 Features
- Object detection using YOLO
- Custom labeled dataset
- Image classification and detection
- Deep learning-based training

---

## 🛠️ Tech Stack
- Python
- YOLO (You Only Look Once)
- OpenCV
- Deep Learning

---

## 📁 Project Structure
- app.py → main detection script
- data.yaml → dataset configuration
- dataset/ → training & validation data
- runs/ → training outputs (ignored in GitHub)

---

## 🚀 How to Run
 Clone:- https://github.com/kriti0317/YOLO-nlp-object-detection-.git


Install the dependencies 


Run: python app.py
